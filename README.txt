# STOCK PRICE monitoring solution

## Aim
To create solution to monitor real-time NASDAQ stock prices and detect price fall in a certain day.
Whenever such event occurs there is high likelihood that a price will go up until a trading day is closed.

## Installation
1. Copy - paste solution to a local computer.
2. Create the .env file with your credentials and email address.
3. Enable App password in the Gmail settings
4. Turn on Gmail notifications

## Results
Year return on investment 32%. Words of cautions - this approach is a day trading.

## Real-time deployment
To deploy the solution any cloud solution can be used - AWS, GCP, PythonAnywhere

## Logs
For validation purpose historical events are collected in app.log file.
